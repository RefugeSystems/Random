identified

  • prefix
  • pre
  • length
)

Defined in index.js:1087

Create a unique identifier following a pattern; [prefix] : [string.length = pre] [Timestamp] [length? string.len = remaining]

Example:
Random.identified('example', 2, 3)

"example:Ha1527351401496iUo"

Parameters:

  • prefix String

    The string to prefix the identifier with before the colon (':'). Defaults to 'id'.

  • pre Number

    Length of the randomly generate string before the timestamp. Defaults to 10.

  • length Number

    The total desired length of the randomized section of the identifier. Fills in after the time stamp if additional length is needed.
